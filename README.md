# Sistema-de-reserva-de-voo

### Problema

    Enunciado: Crie um sistema simples de reservas de voos em C# que não dependa de bibliotecas externas. 
O sistema deve permitir que os clientes visualizem voos disponíveis, escolham assentos, façam reservas e recebam confirmações. Além disso, o sistema deve permitir que a equipe da companhia aérea gerencie voos, registre informações sobre passageiros e faça alterações em reservas, se necessário. O sistema deve ser implementado no console e fornecer uma experiência amigável tanto para os clientes quanto para a equipe da companhia aérea.
