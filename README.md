# Sistema-de-reserva-de-voo
## Problema
    Crie um sistema simples de reservas de voos em C# que não dependa de bibliotecas externas. 
O sistema deve permitir que os clientes visualizem voos disponíveis, escolham assentos, façam reservas e recebam confirmações. Além disso, o sistema deve permitir que a equipe da companhia aérea gerencie voos, registre informações sobre passageiros e faça alterações em reservas, se necessário.

## Resolução do problema
#### Oq os clientes deverão fazer no sistema:
- Visualizar voos disponiveis,
- escolher assentos, 
- fazer reservas,
- receber confirmações

#### Oq o sistema deverá permitir a equipe da companhia aéria fazer:
- Gerenciar voos,
- registrar informações sobre passageiros,
- fazer alterações em reservas.

## Como atingir os pontos acima?
- Ter uma parte somente para clientes
- Outra parte somente para a agência aéria
- Os clientes terão informações sobre os voos
- A agência poderá enviar mensagens para as pessoas que subirão no avião
